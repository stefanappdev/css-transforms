css transforms translation example

translation allows for the movement of an object, in a vertical (y) or horizontal (x) direction

vertical translation


horizontal translation


To perform translation in css - transform:translate(x,y). NB:Positive values move down and to the right. Negative values move up and to the left.

Grid and flexboxes are not affected by using translations on objects

css transforms Scale example

Scale allows us to grow or shrink the sizes of elements. Scale is a unitless property.Scale is implemented using scale(n), where n is a scale factor. Or by using scale(x,y) which defines a horizontal and vertical scale. Scale affects an element and its descendants.


css transforms rotations example

Rotations allow for elements to be rotated at a certain point in space. We typically use the deg unit for rotation, short for degrees but other units exist such as turn. The turn unit represents how many turns the element should make. 1 turn is equal to 360 degrees. It's obscure, but well-supported; the turn unit goes all the way back to IE 9! It is implemented like: transform:rotate(x deg); x being the number of degrees.
